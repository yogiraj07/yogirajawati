README : 
Page Rank Algorithm is implemented in JAVA

****************************************************
Prerequisite :
1) install jdk1.7.0_45 
2) Eclipse Helios

****************************************************
How to run code : 
1) Import the project "PageRank" in the eclipse
2) Open PageRank->src->com.pagerank.CalculatePageRank->    CalculatePageRank.java
3) Run the JAVA program
4) wt2g_inlinks.txt (input file) is already present in the project
****************************************************
Time for execution of the program : - 30 seconds
